#subscribe(**options) ⇒ Hash

订阅标签

可使用 reg id 或 alias 的方式订阅标签

Examples:

subscribe(reg_id: 'abc', topic: 'beijing')
subscribe(alias: 'abc', topic: 'beijing')
subscribe(alias: 'abc,def,ghi,jkl', topic: 'beijing')
